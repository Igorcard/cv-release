# PROJECT.md

## Product Name

Resume Builder

## Product Summary

A platform that generates professional PDF resumes using LinkedIn profile data and adapts them to specific job opportunities.

## Core Capabilities

The product should support:

- Receiving LinkedIn profile data.
- Normalizing professional profile information.
- Creating resume content.
- Improving weak resume descriptions.
- Analyzing job descriptions.
- Matching candidate profile to job requirements.
- Generating PDF resumes.
- Suggesting missing information.
- Suggesting LinkedIn profile improvements.

## Default Resume Model

The default model is `ATS Premium`.

This model balances professional appearance and hiring-system compatibility.

## Supported Resume Models

- `ATS Premium`: default model.
- `ATS Simple`: maximum parsing compatibility.
- `Visual Modern`: better for direct sharing.
- `Complete Package`: generates more than one version.

## Success Criteria

The product is successful when it can:

- Produce a resume that is clear and professional.
- Preserve factual accuracy.
- Highlight real strengths.
- Adapt content to a specific job.
- Avoid empty or irrelevant sections.
- Generate a clean PDF suitable for job applications.
